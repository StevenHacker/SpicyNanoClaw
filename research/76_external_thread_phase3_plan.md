# External Thread Phase 3 Plan

## Purpose

Phase 3 shifts the collaboration program from:

- broad repo coverage

to:

- implementation-facing synthesis
- bounded SNC utility landing
- longer-horizon architecture capture for future custom Claw variants

The rule for this phase is simple:

- no packet exists just to read more code
- every packet must either
  - unlock `SNC Milestone 1`, or
  - convert existing deconstruction into reusable architecture leverage

## Current Read

What is already true:

- the first large OpenClaw/CC breadth wave is accepted
- the main SNC design packets are accepted
- the first Phase-3 utility/synthesis packet wave is now mostly accepted
- the current program bottleneck is no longer "find the seams"
- the bottleneck is now:
  - turn seams into bounded code
  - turn donor notes into reusable architecture doctrine

That means Phase 3 should run on two rails at once:

1. `SNC Milestone 1`
2. `future custom-Claw architecture`
